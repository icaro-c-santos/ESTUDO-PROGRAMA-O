FORMATAÇÃO DE CODIGO

Fonte do Codigo:

Para garantir melhor legibilidade de codigos escritos no html, utilizamos uma formatação de fonte especifica atraves da tag <code>

Exemplo no codigo: <code> if(num>n){printf("n");}<code>

Exemplo no visual: if(num>n){printf("n");}

Observação: Perceba a diferença entre os espaçamentos, e qualidade das letras e simbolos:

Com code: if(num>n){printf("n");}

Sem code: if(num>n){printf("n");}

Manter Identação do Codigo:

Quando queremos escrever um codigo e manter a identação dele usamos a tag <pre>

Exemplo sem a tag: if(num>n){ print("n"); }else{ printf("ERRO!"); }

Exemplo com a tag: 

        if(num>n){
            print("n");
        }else{
            printf("ERRO!");
        }

Observação: Ambos os codigos foram escrito na forma identada porém sem o uso da tag <pre> o html desconsidera os esapços do mesmo.

Citação Curta:

Para fazer uma citação simples utilizamos a tag <q>

Exemplo no codigo: Citando a palavra <q>teste</q> dentro de um texto.

Exemplo no visual: Citando a palavra teste dentro de um texto.

Citação Completa:

Para fazer uma citação simples utilizamos a tag <blockquote>

Exemplo no codigo: Segundo David Ribeiro: <blockquote cite = "https://www.pensador.com/frases_de_programador/">Mesmo que eu seja um programador, na vida, não acredito em lógica condicional. Mesmo conseguindo escrever o algoritmo, jamais conseguiremos executá-lo e explorar os dois argumentos, pois o tempo não retrocede

Exemplo no visual: Segundo David Ribeiro:

Mesmo que eu seja um programador, na vida, não acredito em lógica condicional. Mesmo conseguindo escrever o algoritmo, jamais conseguiremos executá-lo e explorar os dois argumentos, pois o tempo não retrocede

Observação: Podemos também colocar um link para o texto original, usando o parâmetro cite dentro da tag.

Abreviação:

Para fazer uma abreviação usamos a tag <abbr>

Exemplo no codigo: Abreviando a palavra <abbr title ="Cadastro Nacional de Produtos">CNP</abbr>neste texto.

Exemplo no Visual: Abreviando a palavra CNP neste texto.
