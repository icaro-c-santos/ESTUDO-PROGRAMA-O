FORMATAÇÃO DE TEXTO

Dando Destaque/Negrito:

Para dar destaque a algun trecho do texto usamos a seguinte tag: <strong>

Exemplo no Codigo: Destacando a palavra <strong>teste</strong> nesta frase.

Exemplo no Visual: Destacando a palavra teste nesta frase.

Dando Ênfase/Italico:

Para dar ênfase a uma palavra em um texto usamos a seguinte tag: <em>

Exemplo no Codigo: Destacando a palavra <em>teste</em> nesta frase.

Exemplo no Visual: Destacando a palavra teste nesta frase.

Texto Marcado/Marca-Texto:

Para marcar uma palavra em um texto usamos a seguinte tag: <mark>

Exemplo no Codigo: Marcando a palavra <mark>teste</mark> nesta frase.

Exemplo no Visual: Marcando a palavra teste nesta frase.

Observação: A cor padrão do marca-texto é amarela todavia ela pode ser modificada usando CSS.

Texto Pequeno/Letras pequenas

Para diminuir uma palavra em um texto usamos a tag: <small>

Exemplo no Codigo: Diminuindo a palavra <small>teste</small> nesta frase.

Exemplo no Visual: Diminuindo a palavra teste nesta frase.

Observação: A diminuição é quase imperceptível comparada ao tamanho padrão da fonte.

Texto Deletado/Riscado

Para marcar uma palavra como excluída usamos a tag: <del>

Exemplo no Codigo: Marcando a palavra <del>teste</del> como excluida nesta frase.

Exemplo no Visual: Marcando a palavra teste como excluida nesta frase.

Texto Inserido/Sublinhado

Para marcar uma palavra como inserida, ou seja, que ela foi adicionada depois, usamos a tag: <ins>

Exemplo no Codigo: Marcando a palavra <ins>teste</ins> como inserida.

Exemplo no Visual: Marcando a palavra teste como inserida.

Texto Sobrescrito/Acima

Para marcar uma palavra como sobrescrita usamos a tag: <sup>

Exemplo no Codigo: Marcando a palavra <sup>teste</sup> como sobrescrita.

Exemplo no Visual: Marcando a palavra teste como sobrescrita.

Texto Subscrito/Abaixo

Para marcar uma palavra como subscrita usamos a tag: <sub>

Exemplo no Codigo: Marcando a palavra <sub>teste</sub> como subscrita.

Exemplo no Visual: Marcando a palavra teste como subscrita.
