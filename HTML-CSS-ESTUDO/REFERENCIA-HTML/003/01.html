Listas

Lista Ordenada:

Para adcionar uma lista em html usamos a tag <ol>

Exemplo no Codigo:

    <ol type="1" start="1">
    <li>Tarefa 01</li>
    <li>Tarefa 02</li>
    <li>Tarefa 03</li>
    <li>Tarefa 04</li>
    </ol>

Exemplo no Visual:

  1. Tarefa 01
  2. Tarefa 02
  3. Tarefa 03
  4. Tarefa 04

Type: O atributo type define o tipo de seletor, 1 é numeros, A é letras, I é romanos e a é letras pequenas.

Start: O atributo start é apartir de qual elemento a contagem vai iniciar.

Observação: Não é obrigatorio o fechamento do <li>

Lista Não Ordenenada

Para criar uma lista não ordenada usamos a tag <ul>

Exemplo no codigo: 

    <ul type="circle">
    <li>Tarefa 01</li>
    <li>Tarefa 02</li>
    <li>Tarefa 03</li>
    <li>Tarefa 04</li>
    </ul>

Exemplo no Visual:

  • Tarefa 01
  • Tarefa 02
  • Tarefa 03
  • Tarefa 04

Type: Definie o tipo de seletor, circle,square,disc são as opções, se nenhum for definida o padrão é disc.
