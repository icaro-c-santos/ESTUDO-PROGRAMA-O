Pseudos Classe:

Uma pseudo-classe CSS é uma palavra-chave adicionada às declarações de um seletor após um sinal de dois pontos e especificam um estado especial de um elemento. Existem várias pseudo-classes para estilos:

  • :visited - Qunado um link já tiver sido clicado.
  • :hover - Quando o mouse estiver sobre o elemtno.
  • :active - Quando ativamos o elemento. Por exemplo, quando clicamos em um link e não soltamos o botão do mouse.
    Observação: Active funciona apenas enquanto o link (neste caso) está sendo clicado, mas muda novamente com a liberação do botão do mouse.
  • :checked - Representa um elemento radio, checkbox ou option em um select que está marcada ou alternado para um estado ligado.
  • :empty - Representa qualquer elemento que não tenha filhos. Aqui, filhos podem ser outros elementos ou blocos de texto (incluindo espaços em branco)
  • :focus - A mesma coisa do hover poré funciona não so com o mouse mais também com o TAB.
    Observação: Importante usar os dois.
